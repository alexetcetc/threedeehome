# Three Dee Home

A personal 3D digital library space built with A-Frame. Walk through your virtual reading room.

## Quick Start

Open `index.html` in a browser, or serve it locally:

```bash
# Python 3
python -m http.server 8000

# Then visit http://localhost:8000
```

Or deploy to GitHub Pages for free hosting.

## Controls

- **Click** - Enable mouse look
- **WASD** - Walk around
- **Mouse** - Look around

## Customization Ideas

- Add book cover textures (replace colors with `src="bookcover.jpg"`)
- Pull book data from Kindle exports or Goodreads
- Add clickable books that show details
- Multiple rooms connected by doorways
- Ambient music

## Tech

Built with [A-Frame](https://aframe.io/) - a web framework for building 3D/VR experiences.
